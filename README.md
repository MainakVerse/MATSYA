# Shader Hero Section

A beautiful Next.js application featuring animated shader backgrounds and scrolling text effects.

## Features

- Interactive shader background with lighting effects
- Animated scrolling text with smooth transitions
- Responsive design with modern UI components
- Built with Next.js 15, TypeScript, and Tailwind CSS

## Getting Started

1. Install dependencies:
\`\`\`bash
npm install
\`\`\`

2. Run the development server:
\`\`\`bash
npm run dev
\`\`\`

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Technologies Used

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Radix UI components
- Custom shader effects

## Project Structure

- `/app` - Next.js app router pages and layouts
- `/components` - Reusable React components
- `/lib` - Utility functions and helpers
- `/hooks` - Custom React hooks

## Build and Deploy

\`\`\`bash
npm run build
npm start
\`\`\`

The application is ready for deployment on Vercel or any other hosting platform.
